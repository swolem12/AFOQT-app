Want an APK File?

If you need a traditional Android APK file instead of the PWA installation above, you have several options:

Option 1: PWABuilder (Easiest)

  1. Visit PWABuilder.com
  2. Enter: https://swolem12.github.io/AFOQT-app/
  3. Click "Package for Android"
  4. Download your APK

Option 2: Bubblewrap CLI (Advanced)

  1. Install Bubblewrap: npm install -g @bubblewrap/cli
  2. Run our script: ./generate-apk.sh
  3. Follow the prompts
  4. Get your signed APK

For detailed APK generation instructions, see APK-SETUP.md
