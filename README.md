# AFOQT-app
Retro terminal study app for AFOQT preparation

üöÄ **Live Demo:** [https://swolem12.github.io/AFOQT-app/](https://swolem12.github.io/AFOQT-app/)

## Features
- **4 Complete Subjects:** Math (27 topics), Verbal, Reading Comprehension, Physical Science
- **Practice & Test Modes:** Choose between learning mode with instant feedback or test mode simulation
- **RPG Character System:** Level up your character by answering questions correctly
- **Retro Terminal UI:** Tron-style CRT effects with cyan glow and scanlines
- **Dynamic Responsive Design:** Scales to fit any browser or device
- **Offline-capable PWA:** Install and use completely offline
- **Player Profiles:** Track your progress with session history
- **Procedurally Generated Questions:** Unlimited practice with dynamic question generation
- **Web Audio Sound Effects:** Immersive retro beeps and boops

## Usage

### Desktop/Browser
1. Visit [https://swolem12.github.io/AFOQT-app/](https://swolem12.github.io/AFOQT-app/) in any modern browser
2. No installation or build step required
3. Works completely offline after first visit

### Mobile Installation (Recommended)

**Install as PWA (Progressive Web App):**
1. Open [https://swolem12.github.io/AFOQT-app/](https://swolem12.github.io/AFOQT-app/) in Chrome on your Android phone
2. Tap the menu (‚ãÆ) and select **"Add to Home Screen"** or **"Install App"**
3. The app icon will appear on your home screen
4. Launch from your home screen - runs in fullscreen mode like a native app
5. Works fully offline after installation

**Want an APK file instead?**  
See [APK-SETUP.md](APK-SETUP.md) for detailed instructions on converting this PWA to an Android APK file using PWABuilder, Bubblewrap, or Android Studio.
