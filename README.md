# AFOQT-app
Retro terminal study app for AFOQT preparation

## Features
- 27 Math topics with procedurally generated questions
- Offline-capable Progressive Web App (PWA)
- Player profiles with session history
- Retro Knight Rider aesthetic
- Web Audio sound effects

## Usage

### Desktop/Browser
1. Open `index.html` in any modern browser
2. No installation or build step required
3. Works completely offline

### Mobile (PWA Installation)
1. Open the app in Chrome/Edge/Safari on your phone
2. Tap the menu (⋮) and select "Add to Home Screen" or "Install App"
3. The app will install like a native app
4. Launch from your home screen - works fully offline

## PWA Features (NEW!)
✅ Installable on mobile devices
✅ Offline caching via Service Worker  
✅ App icons for home screen
✅ Standalone mode (no browser UI)
✅ Theme color matching retro aesthetic
